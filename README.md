# 🎓 College Assistant Chatbot

An AI-powered chatbot for college FAQs with voice input support and a warm, modern interface.

## ✨ Features

- 🤖 Natural language understanding for college queries
- 🎤 Voice input support (Chrome/Edge)
- 🌐 Multi-language toggle (English/Hindi beta)
- 📱 Fully responsive design
- 💬 Real-time chat interface
- 🏷️ Quick suggestion chips

## 🛠️ Tech Stack

- **Backend:** Flask (Python)
- **ML/NLP:** scikit-learn, NLTK
- **Frontend:** HTML5, CSS3, JavaScript
- **Voice:** Web Speech API

## 🚀 Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/mansaakohli15/college-chatbot.git
cd college-chatbot
2. Install dependencies
bash
pip install -r requirements.txt
3. Train the model
bash
python train_chatbot.py
4. Run the application
bash
python app.py

5. Open your browser
Navigate to http://127.0.0.1:5000
